## ============================================================
## KEY BUSINESS RULES
## ============================================================

# RAG Vector Search:
- Chỉ tìm trong chunks của ACTIVE VERSION (Chapter.CurrentVersionId)
- Worldbuilding + Characters: tìm tất cả (không theo version)
- TopK: 3 chapters + 2 worldbuilding + 2 characters

# AI Token & Analysis:
- Chat RAG: chỉ trừ UsedTokens — KHÔNG trừ UsedAnalysisCount
- Analyze: trừ cả UsedTokens + UsedAnalysisCount
- Rewrite: chỉ trừ UsedTokens

# Chapter Version:
- Max 20 versions/chapter — auto-prune oldest non-pinned
- Pinned versions: không bao giờ bị auto-delete
- Create version: snapshot current content (không tạo rỗng)
- Chunks xóa cascade khi version bị prune

# Encryption (AES-256):
- Mỗi user có DEK riêng
- DEK mã hóa bằng Security:MasterKey
- Encrypted fields: chapter content, titles, characters, worldbuilding, chat messages

# Auto Embed on Save:
- Khi lưu chương → backend tự chunk + embed ngầm
- Worldbuilding/Character: auto embed khi tạo/cập nhật
